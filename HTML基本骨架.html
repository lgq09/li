基本骨架

html: 整个网页
head: 网页头部，存放给浏览器看的代码，列如CSS
body: 网页主体，存放给用户看的代码，列如 图片，文字
title: 网页标题

VS Code 快速生成骨架:

在HTML文件中 用 !(英文) 加 Entrt / Tab 键

注释

在VS Code 中，添加/删除注释快捷键：ctrl+/

图片标签

img src="./" alt="图片注释"
width 图片的宽度
height 图片的高度
title 提示文本 鼠标悬停在图片上显示的文字

路径

相对路径
/（进入到某个文件夹里面）
.表示当前文件所在文件夹 ./
../ 表示上一级文件夹 ../../表示上两级
